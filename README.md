# 📊 BrightLearn Data Analytics – Research Assignment 1

This repository contains **Research Assignment 1** for the *Foundations of Data Analytics & Data Science* course by BrightLearn.

📄 Source document: :contentReference[oaicite:0]{index=0}

---

## 📌 Overview

This assignment introduces **beginner-level students** to core concepts in:

- Data Fundamentals  
- Artificial Intelligence (AI) & Machine Learning (ML)  
- APIs and Developer Tools  
- Data Ethics and Governance  
- Careers in Data  

The objective is to build a **strong conceptual foundation** through research-based learning.

---

## 🧠 Assignment Structure

### 🔹 Section A: Data Fundamentals
Topics covered include:
- Databases, Data Warehouses, Data Lakes, Data Marts  
- Structured, Semi-Structured, and Unstructured Data  
- Data Modeling and Data Mining  
- ETL (Extract, Transform, Load)  
- Data storage formats (CSV, JSON, Parquet, Avro)  

---

### 🤖 Section B: Artificial Intelligence & Machine Learning
Focus areas:
- Artificial Intelligence vs Machine Learning  
- Types of ML (Supervised, Unsupervised, Reinforcement)  
- Deep Learning and Neural Networks  
- Natural Language Processing (NLP)  
- Large Language Models (LLMs) and Generative AI  
- Computer Vision  

---

### 🔌 Section C: APIs, Tools & Developer Concepts
Key concepts:
- APIs and REST architecture  
- API Keys and security  
- JSON data format  
- Prompt Engineering  
- Cloud Computing (IaaS, PaaS, SaaS)  
- Git and Version Control  
- Python and Jupyter Notebooks  

---

### ⚖️ Section D: Data Ethics, Governance & Careers
Includes:
- Data Privacy and GDPR  
- Data Ethics and Bias in AI  
- Data Governance frameworks  
- Roles: Data Analyst, Data Scientist, Data Engineer, AI Engineer  
- Business Intelligence (BI) tools  

---

### ✍️ Section E: Personal Reflection
Students reflect on:
- Key learnings  
- Challenges encountered  
- Career relevance of Data and AI  

---
